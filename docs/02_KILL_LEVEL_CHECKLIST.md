# Kill-Level Feature Checklist

## Non-Negotiable Rules
- Reduce cognitive load
- Remove decisions
- Feel finished
- Be opinionated
- Work instantly

---

## App Foundations
- [ ] React Native app (Expo)
- [ ] Android-first build
- [ ] Minimal design system
- [ ] Fast cold start
- [ ] Graceful error handling

---

## Onboarding (One Screen)
- [ ] One-sentence value proposition
- [ ] Optional context input (“What should your coach know?”)
- [ ] Optional values input
- [ ] Clear skip option
- [ ] No accounts

---

## Home Screen
- [ ] Title: “What do you need right now?”
- [ ] Exactly 3 large coach cards
- [ ] No scrolling
- [ ] Focus Coach visually defaulted

---

## Coaches
### Focus Coach
- [ ] Short, directive answers
- [ ] Forces prioritization
- [ ] Pushes to action

### Decision Coach
- [ ] Structured thinking
- [ ] Forces tradeoffs
- [ ] Stops spirals

### Reflection Coach
- [ ] Slower pace
- [ ] Fewer words
- [ ] Values-based framing

All coaches:
- [ ] Stay strictly in role
- [ ] Use user context
- [ ] Never overwhelm

---

## Chat Experience
- [ ] Instant start
- [ ] Clean, minimal UI
- [ ] Coach name fixed at top
- [ ] Context auto-injected
- [ ] Simple coach switch

---

## Monetization (RevenueCat)
- [ ] Free tier limits enforced
- [ ] Subscription unlocks unlimited chats
- [ ] Persistent context for paid users
- [ ] Calm, respectful paywall copy

---

## Create Your Own Coach (Premium)
- [ ] Duplicate existing coach
- [ ] Edit goal / tone / constraints
- [ ] Same chat UI
- [ ] No blank canvas

---

## Polish
- [ ] Loading states
- [ ] Empty states
- [ ] Friendly error copy
- [ ] No dead ends

---

## Explicit Cuts
- [ ] No marketplace
- [ ] No community gallery
- [ ] No infinite modes
- [ ] No habit tracking
- [ ] No dashboards
